Программы для Android

Здесь собраны программы для Android! Посмотри, может найдёшь что-нибудь полезное!

Вернуться в раздел "Программы для ПК"

Перейти в раздел "Модули Magisk (ROOT)"

Termux

Терминал для Android (Для полного доступа требуется ROOT доступ)

RePainter

Программа, позволяющяя принудительно менять цвета на Android 12 - 13. Требуется либо ROOT доступ, либо установить Shizuku

AIDA64

Утилита FinalWire Ltd. для тестирования и идентификации компонентов персонального компьютера под управлением операционных систем Android, предоставляющая детальные сведения об аппаратном и программном обеспечении.

Total Commander

Простой, но удобный файловый менеджер с возможностью работы сразу в 2-х окнах. Для полной работы потребуется ROOT доступ
