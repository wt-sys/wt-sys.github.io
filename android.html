Termux

Terminal for Android (ROOT required for full access)

RePainter

A program that allows you to force change colors on Android 12 - 13. Either ROOT access is required or Shizuku must be installed

AIDA64

FinalWire Ltd. utility. for testing and identifying components of a personal computer running Android operating systems, providing detailed information about hardware and software.

Total Commander

A simple but convenient file manager with the ability to work in 2 windows at once. ROOT access is required for full operation
